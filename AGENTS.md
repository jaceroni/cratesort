# CrateSuite Project Instructions

## Non-Negotiable UI & Core Logic Rules
- Teal (#428175) = Action color. 
- Orange (#D17D34) = Selection color. 
- Banned Taxonomy: "Pop" is NEVER a valid genre.
- Artist-level genre changes must NEVER cascade to individual tracks.
- Never touch Serato metadata or comments.

## App Dev Working Style
- Never begin building code or prompts without explicit approval on the tradeoffs first.
- Collect all feedback into a full punch list, confirm it, then execute one item at a time.
- Claude Code / Sandbox execution: Use Sonnet for CSS/JS integrations, Opus for complex multi-step debugging pipelines.
- All prompts generated by the system for local use must be output as markdown (.md) files.
